# Registration Methods for Spinal Cord

## Robust Multimodal Registration

Multi-level approach using

* Automatic spinal cord segmentation using mesh propagation.
* Large-scale deformation based on segmentations
* Small-scale deformation using [ANTS](http://stnava.github.io/ANTs/)
